# My-Brand-Cherubin
This is my branding site.
## Features 
##### Admin features:
* login
* create a blog.
* edit a blog.
* delete a blog.
* view queries.
* view a single query.
* delete a single query.
##### User features:
* view a blog.
* view all blog.
* like a blog.
* comment on blog.
* send a query.

## Installation
* install Nodejs
* Clone the repository https://github.com/Cherubln/My-Brand-Cherubin
* npm install to install packages
* npm start to start the server
